# Rdt

Command line Reddit built in Elixir.

Compile:
```
mix deps.get
mix escriptize
```

Usage:
```
    usage:  rdt [COMMAND] [COMMAND_PARAMETERS]

    commands:
	-h  --help								# This message
	-lh	--hot    	?<subreddit> 			# Get hot articles (subreddit optional)
	-c	--comments  <subreddit> <id> 		# Gets a specific article and comments
```
